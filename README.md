# WIOIP
WIOIP atau Weather Information over Internet Protocol adalah suatu software simple yg berguna untuk mendapatkan data cuaca dari internet dan ditulis dalam bahasa Java dengan library JavaFX.

 - JDK (min 1.8.x)
